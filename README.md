# 🔎 Port Scanner CLI

A lightweight and powerful **port scanning tool** built with Python.
Scan any website or IP address to discover open ports and understand active services running behind the scenes ⚡

---

## 🚀 Features

* 🌐 Scan any domain (auto converts to IP)
* 🔍 Custom port range scanning
* ⚡ Fast and efficient socket-based scanning
* ❌ Handles invalid inputs gracefully
* 🧠 Clean and minimal CLI interface

---

## 🛠 Tech Stack

* Python 🐍
* socket (network communication)
